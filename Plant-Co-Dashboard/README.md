Plant Co Performance Analysis (Power BI Dashboard)
## Description:
This project analyzes the Plant Co dataset using an interactive Power BI dashboard to evaluate
across categories and time. The focus is on identifying trends, comparing results, and uncovering
patterns in key business metrics

## Key Insights:
- Performance varies across product categories, highlighting areas of strength and weakness
- Time-based trends reveal periods of growth and decline
- Country-level comparisions show differences in contribution to overall results
- Profitability analysis highlights the relationship between sales value and margins

## Skills Demonstrated:
- KPI and trend analysis
- Data visualization and dashboard design
- Performance analysis
- Interactive reporting with Power BI
